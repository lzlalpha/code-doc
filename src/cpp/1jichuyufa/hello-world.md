## 第一份代码

通过这样一个示例程序来展开 C++ 入门之旅吧～

注：请在编写前注意开启英文输入法。

C++ 语言

```c++
#include <iostream>  // 引用头文件

using namespace std; // 引入命名空间

int main() {                // 定义 main 函数
  cout << "Hello, world!";  // 输出 Hello, world!
  return 0;                 // 返回 0，结束 main 函数
}
```
